How to install?
===============

```
$ git clone --recursive git@github.com:mahendrakalkura/diffusion.git
$ cd diffusion
$ mkvirtualenv diffusion
$ pip install --requirement requirements.txt
```

How to run?
===========

```
$ cd diffusion
$ workon diffusion
$ python manage.py --bet365
$ python manage.py --williamhill {{ id }}
```
